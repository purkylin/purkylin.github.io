macOS编程

  • titlebar透明， 保留关闭最大化等按钮
    Ok, I found the solution. You have to create a subclass of ‘NSWindowControllerand in thewindowDidLoad` method, you have to put this code:

self.window!.titlebarAppearsTransparent = true
to make the whole thing work, you have to activate “Full Size Content View” in Interface Builder.

  • 管理开机启动项
    选择“登陆项”选项卡，我们就能看到当前的开机启动项了，勾选不需要的开机启动项，然后点击“－”按钮删除就行，如果要增加开机启动想，点击“＋”按钮，选择需要开自己启动的项目就行

  • Ruby源修改

  • 强杀应用 Command + ALT + ESC

  • launchpad残留图标怎么删除？
    按住option键图标的左上角会出现删除的x号，点击x号就可以删除啦
  • .ssh 免登陆
    修改用户的%h/.ssh/authorized_keys, 如果没有手动新建，追加你的公钥即可
  • 在 macOS Sierra 中，开启 Finder，并使用 Shift + Command + .组合键，就能够让系统隐藏文件显示出来。再按一次，就能恢复原貌，十分简便。
    • sublime 正则替换 使用$1, $2, …

  • Finder快捷键

    • Command H Hide Application
    • Command option H Hide other applications
    • ⌘ ⇧ A Jump to /Applications
    • ⌘ ⇧ U Jump to /Applications/Utilities/
    • ⌘ ⇧ D Jump to ~/Desktop
    • ⌘ ⇧ H Jump to Home Folder ~/
    • ⌘ + ↑ Move up one directory
    • ⌘ + [ go back
    • ⌘ + ] go forward
    • ⌘ + R Show original
    • ⌘ + L Make Alias
    • control + y cancel

  • install cask
